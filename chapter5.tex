\section{Cost Estimation using COCOMO-II Model}

\subsection*{Step 1: Introduction}
The COCOMO-II (Constructive Cost Model II) is used to estimate the effort, development time, and cost required for software development. It is suitable for modern applications involving web technologies, APIs, and machine learning systems.
\vspace{0.4cm}

For the InferX-ML system, the Object Point method of COCOMO-II is used, as it is based on UI screens, APIs, and system components.
\vspace{0.4cm}

\subsection*{Step 2: Project Type}
\begin{itemize}
    \item \textbf{Project Type:} Organic
    \item \textbf{Reason:}
    \begin{itemize}
        \item Small development team (3 members)
        \item Moderate system complexity (Web + ML integration)
        \item Use of modern frameworks and APIs
    \end{itemize}
\end{itemize}
\vspace{0.4cm}

\subsection*{Step 4: Total Object Points}
\[ UOP = 22 + 35 + 105 = 162 \]

\subsection*{Step 5: Adjusted Object Points}
Assuming 20\% reuse:
\[ NOP = 162 \times (1 - 0.20) = 129.6 \approx 130 \]

\subsection*{Step 6: Effort Estimation}
Productivity Rate = 9.5 Object Points / Person-Month
\[ Effort = \frac{130}{9.5} \approx 13.7 \approx 14 \text{ Person-Months} \]

\subsection*{Step 7: Development Time}
Team Size = 3 Members
\[ Time = \frac{14}{3} \approx 4.5 \text{ Months} \]

\subsection*{Step 8: Cost Estimation}
\begin{itemize}
    \item[a)] \textbf{Development Cost:} $4.5 \times 25,000 \times 3 = \text{Rs. } 3,37,500$
    \item[b)] \textbf{Hardware \& Software Cost:} $\text{Rs. } 45,000$
    \item[c)] \textbf{Miscellaneous Cost:} $\text{Rs. } 10,000$
\end{itemize}

\subsection*{Step 9: Total Cost}
\[ TotalCost = 3,37,500 + 45,000 + 10,000 = 3,92,500 \]
\textbf{Total Estimated Cost $\approx$ Rs. 3.9 Lakhs}
\vspace{0.4cm}

\subsection*{Step 10: Conclusion}
The COCOMO-II estimation shows that the InferX-ML system requires approximately \textbf{14 person-months} of effort, with a development duration of around \textbf{4.5 months} for a team of 3 members. The total estimated cost of development is approximately \textbf{Rs. 3.9 Lakhs}, making it a feasible and cost-effective software solution.
